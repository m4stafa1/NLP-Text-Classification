# NLP-Text-Classification
This project focuses on building an intelligent Text Classification system using multiple Natural Language Processing (NLP) techniques and comparing their performance



## Overview
This project implements text classification using:
- Machine Learning
- Deep Learning
- Transformers (Hugging Face)



## The project was implemented through several phases:

Data Preprocessing and Cleaning
Feature Extraction using TF-IDF
Traditional Machine Learning Models
Deep Learning using TensorFlow/Keras
Transformer-based Models using Hugging Face
Models Used
Linear SVM
Neural Network (ANN)
DistilBERT (Fine-Tuned)

## Technologies & Libraries
Python
Scikit-learn
TensorFlow / Keras
Hugging Face Transformers
PyTorch
NLTK
Streamlit
Workflow
Text Cleaning and Preprocessing
Tokenization and Lemmatization
TF-IDF Vectorization
Machine Learning Classification
Deep Learning Classification
Transformer Fine-Tuning
Deployment using Streamlit

## Results

The project compares traditional ML models with Deep Learning and Transformer-based architectures.
Fine-tuned DistilBERT achieved the best performance due to its contextual understanding of language.

## Deployment

A Streamlit web application was developed to allow users to input custom text and receive predicted categories in real time.

## Project Goal

The main goal of this project is to explore and compare different NLP approaches for text classification tasks and understand how modern Transformer models outperform traditional techniques in contextual language understanding.
